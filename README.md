# Historical Decision Simulator Example

This example demonstrates a basic text-based historical decision-making simulator. It presents a scenario where the user, as a historical figure, must make a critical decision. The chosen action influences the simulated state (e.g., resources, morale, reputation) and leads to a specific outcome, illustrating the core concept of interactive decision-making in a safe, simulated environment.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open a terminal or command prompt.
3. Navigate to the directory where you saved `main.py` and run: `python3 main.py`
